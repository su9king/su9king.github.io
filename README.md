# su9king.github.io

> Software is Art. — A film by Min-cheol Shin.

A cinematic personal site that argues, in eight scenes, that software is a
form of visual art. Click anywhere — or press any key — to advance.
`←` / `Backspace` to step back.

## Scenes

1. **Title** — *Software is Art.*
2. **Prologue** — software as a new canvas of culture
3. **Act I** — Aesthetics of Interaction (cursor trail)
4. **Act II** — Visualizing the Invisible (live topology)
5. **Act III** — The Visual Structure of Code (typed in real time)
6. **Act IV** — A New Paradigm of Creation (generative flow field)
7. **Epilogue**
8. **End Credits** — who I am, what I'm doing, where to reach me

## Stack

Pure HTML / CSS / Canvas. No build step, no dependencies beyond Google Fonts.
Open `index.html` in a browser, or visit
[su9king.github.io](https://su9king.github.io).
